Web Development

Web Development refers to the creation, building and maintenance of a website. It is made up of two words "WEB" and "DEVELOPMENT". "WEB" means a webpage or website and "DEVELOPMENT" refers to the building of application from scratch.
Web Development is of 3 types: 1. Frontend Development 2. Backend Development 3. Full Stack Web Development
1.Frontend Development:
It is a part of the website which is directly visible to the user to which he can intract with. It is also known as client side of the application as it is visible to the user.
2. Backend Development:
It is a part of website which is not visible to the user and he cannot intract with it. It is also known as server side of the website which is used to process, store, and arrange the data.
3. Full Stack Web Development:
It refers to the ability to work on both frontend and backend side of a web application.
