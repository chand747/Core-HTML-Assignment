Full Stack Web Development tech stack

HTML
HTML stands for Hyper Text Markup Language It is used for creating web pages and layouts.
CSS
CSS stands for CasCading Style Sheet. It is a style sheet language and it is basically used for styling a web pages.
JavaScript
JavaScript is bacically a scripting language. It is used to create and control dynamic websites and contents. JavaScript is easy to learn.
Node.js
Node.js is a open source platform, JavaScript is a runtime environment that executes outside of a browser.
Express.js
Express.js is a small framework that works on the top of Node.js web server functionality to simplify its APIs and add helpful new features.
MongoDB
MongoDB is an open source NoSQL database management program. NoSQL (Not only SQL) is used as an alternative to traditional relational database.
React.js
React.js is a open-source frontend JavaScript library for building user interface based on components.
